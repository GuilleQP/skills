# 🧠 skills

[![Skills](https://img.shields.io/badge/skills-2-blueviolet)](#-whats-inside)
[![Spec](https://img.shields.io/badge/spec-Agent%20Skills-informational)](https://agentskills.io/)

A collection of self-made **agent skills** — bite-sized instruction packs that teach an AI agent how to do one thing *really* well.

## 📦 What's inside

| Skill | What it does |
|-------|--------------|
| [`eli5`](eli5/SKILL.md) | Explains hard concepts with one faithful, everyday analogy. Skips any domain specific argot. |
| [`gitmoji`](gitmoji/SKILL.md) | Writes a commit message prefixed with the right [gitmoji](https://gitmoji.dev), when you ask for one. |

## ⚡ How to use

The agent picks the right skill automatically based on what you ask. Just talk naturally:

> "Wait, ELI5 how vector search works."

…and the `eli5` skill kicks in. ✨
